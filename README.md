# **GoBang**

使用Qt5.7.0的五子棋小软件，支持人人对战，人机对战，局域网联机对战

学习Qt后的一个练手项目，完成自己在命令行下的五子棋升华^_^

人机对战使用基于评分表的策略来完成电脑下子，算不上AI吧。。但是电脑的棋力还是很高的。

在无禁手黑棋先行的情况下，这个评测表对于电脑白棋的棋力会稍弱一些

局域网联机对战使用的QUDP实现

还有很多bug和细节未完善。。确切说烂尾- -
